# 🚀 ВНИМАНИЕ! Переименование проекта

## MK Launcher → UMCL

Проект **MK Launcher** официально переименовывается в **UMCL**.

Новое название:

# UMCL

**UMCL** расшифровывается как:

> Unofficial Minecraft Client Launcher

Переименование сделано для того, чтобы название лучше отражало назначение проекта и было более понятным для новых пользователей.

## Что изменилось?

- ✅ Новое название проекта
- ✅ Обновление названий файлов и пакетов
- ✅ Обновление интерфейса
- ✅ Подготовка к дальнейшему развитию проекта

## Что осталось?

Все основные функции лаунчера сохраняются:

- Запуск Minecraft Java Edition
- Управление версиями
- Поддержка Linux
- Обновления проекта

## Важно

UMCL является **неофициальным лаунчером** и не связан с Mojang Studios или Microsoft.

Minecraft является товарным знаком Mojang Studios.
